# Proyecto-Intermodular-1-SMR - Infraestructura informática para CreaDigital SMR
## Presentación del proyecto
Este proyecto consiste en diseñar y configurar la infraestructura informática completa para una agencia de marketing digital llamada "CreaDigital SMR". Hay que darle a la empresa de una red segura, equipos adecuados para diseño gráfico y tareas administrativas, y un sistema para compartir archivos pesados. Está pensado para una oficina de 5 empleados. He utilizado tecnologías como Windows 11, packet tracer/draw.io para el diseño de red, y he investigado soluciones Cloud en AWS. Desarrollando esto he aprendido cómo se integran los sistemas, las redes y el hardware en un entorno real, dándome cuenta de que todo está conectado.
## Reflexión personal
Durante este proyecto he aprendido a ver la informática desde la perspectiva de una empresa y no solo de un usuario. Lo que mejor se me ha dado ha sido el diseño de la red y la selección del hardware, ya que me gusta investigar componentes. Lo que más me ha costado ha sido entender los costes de la nube, ya que hay muchos servicios distintos. Para mejorar, quiero seguir practicando la configuración de permisos de red y crear máquinas virtuales de forma más fluida. Mi próximo paso será intentar montar un pequeño servidor NAS casero este verano para seguir practicando.
